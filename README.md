# ⚾ 지존투수 키우기 (도트 로그라이크 투수 키우기)

2D 도트 감성의 웹/모바일(9:16) **투수 키우기 로그라이크** 게임. 순수 `HTML/CSS/Vanilla JS` 단일 파일(`index.html`)로 동작합니다.

## 플레이 방법
- `index.html` 을 브라우저로 열면 바로 실행됩니다. (별도 설치/빌드 없음)

## 주요 기능
- **투수 생성**: 주사위로 구속/제구(각 40~60 랜덤) + 17종 특성 중 1개 랜덤 부여
- **투구 배틀**: 빠른볼(구속vs파워) / 변화구(제구vs선구안) 즉시 판정
  - 기본 분포 — 아웃 70%(땅볼25·플라이25·삼진20) / 출루 30%(단타15·2루타4·3루타1·홈런4·4구6)
- **고유 특성 17종**: 라이징 패스트볼·마구의 장인·볼링공 구위·강심장·클로저 본능·싸움닭·맞춰잡기의 달인·스타성 등
- **슬롯머신 보상**: 아웃마다 등급별(일반~전설) EXP/GOLD 획득
- **주간 루프**: 화~일 경기 / 월요일 랜덤 상점(13종 아이템) + 성장 훈련
- **난이도 스케일링**: 일차에 따라 타자 강화 + 5% 확률 '각성' 엘리트 타자
- 효과음(WebAudio 합성) · 타격 임팩트 연출 · 통산 아웃/방어율 기록
